Языковые раскладки Последняя языковая раскладка открывается:
  1. При открытии клавиатуры
  2. При нажатии клавиши с кодом -2 на любой раскладке, которая не является последней языковой раскладкой
Основной способ перехода между языковыми раскладками – это коды переключения языков.
Также можно открыть раскладку определённого языка с помощью значений $$ru, $$en и т. д. (см. атрибуты android:keyboard и android:longKeyboard). Открытая таким образом раскладка – временная: клавиша с кодом -2 на открытой таким образом раскладке будет возвращать к последней языковой раскладке. Неязыковые раскладки
  • symbol – символьная раскладка

    Открывается:
    1. По нажатию клавиши с кодом -2 на языковой раскладке
    2. По нажатию клавиши Shift на дополнительной символьной раскладке
    3. С помощью значения $$symbol (см. атрибуты android:keyboard и android:longKeyboard)
  • symbol2 – дополнительная символьная раскладка

    Открывается:
    1. По нажатию клавиши Shift на символьной раскладке
    2. С помощью значения $$symbol2 (см. атрибуты android:keyboard и android:longKeyboard)
  • edittext – раскладка редактирования текста

    Открывается:
    1. По долгому нажатию клавиши Shift
    2. По нажатию клавиши с кодом -611 или -619
    3. С помощью значения $$edittext (см. атрибуты android:keyboard и android:longKeyboard)
  • smile – раскладка смайликов

    Открывается:
    1. По долгому нажатию клавиши Enter
    2. По нажатию клавиши с кодом -610 или -620
    3. С помощью значения $$smile (см. атрибуты android:keyboard и android:longKeyboard)
  • num – числовая раскладка

    Открывается:
    1. Если атрибут type тега <input>, который создаёт поле ввода текста, имеет значение number (<input type="number">). Тестовый пример для проверки:
    2. По нажатию клавиши с кодом -612 или -621
    3. С помощью значения $$num (см. атрибуты android:keyboard и android:longKeyboard)
  • calc – раскладка калькулятора
Скрытые раскладки Конкретные скрытые раскладки можно открывать с помощью атрибутов android:keyboard и android:longKeyboard. Также существует код -628.
