У понятия «раскладка» есть 2 значения:
  • В узком смысле — непосредственно сам xml файл.
  • В широком смысле — xml файл вместе со всеми файлами, пути к которым в нём указаны, т. к. если эти файлы не будут существовать, раскладка не будет работать так, как было задумано её создателем.
Теги Атрибуты Форматы файлов
<Key /> android:keyIcon png, jpg, jpeg
android:keyboard
android:longKeyboard 		xml
android:openTextFile
android:longOpenTextFile 		txt
<includeReplace /> file
Соответственно, если ваш файл находится прямо в этой папке, путём к нему будет просто его имя (вместе с точкой и форматом после неё). Если же, например, у вас в корневой папке есть папка Папка1, а в ней файл Файл1.txt, то путём к последнему будет значение Папка1/Файл1.txt и далее по такому же принципу.
